How to run this app properly
You need to serve this HTML file through a local server. Here are 3 easy ways:
Option 1 — VS Code (recommended)
Install "Live Server" extension → Right-click the HTML file → "Open with Live Server"
Option 2 — Python (built-in)
# Open terminal in the folder containing this file, then run:
python -m http.server 8080
# Then open: http://localhost:8080/tdmh_maintenance_hub.html
Option 3 — Node.js
npx serve .
# Then open the URL it shows you
💡 For production use, upload this file to any web hosting — Netlify, Vercel, or your hospital's internal server.
